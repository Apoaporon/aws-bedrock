# aws-bedrock

### フロー
1. 1つのクエリ（入力プロンプト）から複数の類似クエリを作成する
2. 以下のような辞書を作成
    - {検索クエリ：ナレッジ検索結果}
3. 最終的なクエリの作成
4. 解答を得る
